# [typing decorators sucks! here's an easier way (intermediate)](https://youtu.be/_QXlbwRmqgI)

ParamSpec / TypeVar / Callable oh my!  this skips all of that and gives a simple easy-to-reuse decorator (and more!)

## Setup commands

```bash
virtualenv venv/bin/activate
. venv/bin/activate
pip install mypy
```

## Interactive examples

### Bash

```bash
mypy t.py
python3 t.py
```
